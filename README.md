## Libraries Required
- pandas 
- jupyter 
- seaborn 
- scikit-learn 
- keras 
- pytorch 
- pillow
- torchvision

Install above libraries using - **pip install -r requirements.txt**

Download Data from here : https://nasadata.blob.core.windows.net/nasarocks/Data.zip
